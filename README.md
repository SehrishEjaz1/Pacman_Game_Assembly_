# Pac-Man Game — x86 Assembly
A Pac-Man game built in x86 Assembly language with VGA text-mode graphics, animated character movement, keyboard input handling, and screen rendering at low level.

**Course:** Computer Organization & Assembly Language (COAL) | FAST NUCES Lahore
**Developed by:** Sehrish Ejaz

## Overview
A Pac-Man game built in x86 Assembly language using VGA text-mode graphics.

## Features
- Animated Pac-Man character movement
- Multiple ghosts with different colors
- Keyboard input handling
- Score tracking
- Screen rendering at low level

## Tools Used
- x86 Assembly Language
- DOSBox Emulator
